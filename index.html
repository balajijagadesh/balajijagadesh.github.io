Small tools that help you create language-aware wiki text. Hosted on GitHub Pages.

WikiLink Translator

Translate wiki links inside text (works with Wikidata sitelinks). Input and output boxes — editable output and copy button.

Open the translator:

Open Translator
How to use
  • Type text in the left box. Use [[Article]] to mark items to translate.
  • Select source and target languages from the dropdowns and click Convert.
  • Output is editable — then click the Copy button.
